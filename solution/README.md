

# 💰 Money Tracker

## 📌 Overview

Money Tracker is a full-stack web application designed to help users track daily expenses, monitor spending habits, and manage their monthly budget efficiently.

The system provides both **total spending** and **category-wise analysis (amount + percentage)** to give clear financial insights.

---

## 🚀 Features

* ➕ Add and manage daily expenses
* 📂 Categorize expenses (Food, Travel, Bills, etc.)
* 📋 View complete expense history
* 📊 Monthly summary (Total + Category-wise breakdown)
* 🎯 Budget management and alerts
* 🧠 Smart insights (spending patterns & analysis)
* 🔍 Filter expenses by date and category

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/money-tracker.git
cd money-tracker
```

### 2️⃣ Backend Setup

```bash
cd solution/backend
npm install
npm start
```

### 3️⃣ Frontend Setup

```bash
cd solution/frontend
npm install
npm start
```

---

## 📊 How It Works

1. User logs in to the system
2. Adds daily expenses
3. System stores data in database
4. Monthly summary is calculated
5. Dashboard shows:

   * Total spending
   * Category-wise amount
   * Percentage analysis
